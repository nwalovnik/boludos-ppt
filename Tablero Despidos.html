Agregar evento por link de noticia
Pegá el link a una noticia de Clarín, Infobae, Ámbito, etc. El sistema baja la nota, extrae empresa/empleados/fecha/provincia y la suma al registro. Necesita un GitHub token configurado (engranaje arriba a la derecha).

Configuración

Generá uno en github.com/settings/tokens con scope contents: read+write + actions: read+write sobre el repo nwalovnik/boludos-ppt. Se guarda solo en este navegador (localStorage).
